# knowledge document（RAG 知識文件）資料需求

## 來源
- `ADMIN_TEST_DATA_FORMAT.md`：`## knowledge document`
- 參考：`.omo/notepads/rag-data-collection/knowledge-documents.json`
- 補充公開資訊來源方向：鹿谷鄉公所、南投旅遊網、小半天休閒農業區、農業知識入口網、農業易遊網等公開景點介紹。

## API
- `POST /api/admin/knowledge-documents`

## request body
```json
{
  "title": "茶園背景知識",
  "content": "本茶園主要種植青心烏龍，入口區域設有導覽牌。",
  "source": "admin-seed",
  "questId": "11111111-1111-1111-1111-111111111111",
  "locationId": "aaaaaaaa-aaaa-aaaa-aaaa-aaaaaaaaaaaa",
  "spoilerLevel": "GENERAL"
}
```

## 欄位規格
- `title`: string，必填，不可空白，max 255
- `content`: string，必填，不可空白
- `source`: string，必填，不可空白，max 255
- `questId`: UUID，可不填
- `locationId`: UUID，可不填
- `spoilerLevel`: string，必填（無固定值限制）
